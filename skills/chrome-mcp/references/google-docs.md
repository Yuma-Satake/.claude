# Google Docs 操作ナレッジ

Google ドキュメントを Chrome MCP で操作する場合の固有ナレッジ。

## 見出しスタイルの適用

### ショートカットキー

| スタイル | ショートカット |
|---|---|
| Normal text | `Cmd+Alt+0` |
| Heading 1 | `Cmd+Alt+1` |
| Heading 2 | `Cmd+Alt+2` |
| Heading 3 | `Cmd+Alt+3` |

### 適用手順（空行誤適用を防ぐ）

空行と本文行が隣接している場合、座標クリックで空行を選択してしまいやすい。以下の手順で確実に目的行に適用する。

1. 目的行の付近をクリック
2. `zoom` で選択行のテキストを確認（空行か本文行かを判断）
3. 空行の場合 → `key: Down` で次の行へ移動
4. ツールバーのスタイル表示（`zoom` で確認）が期待通りか確認
5. `Cmd+Alt+1`（または 2）でスタイルを適用

### 適用後の確認方法

- **ツールバー**: 左サイドバー上部のスタイルドロップダウンが "Heading 1" / "Heading 2" に変わっていることを確認
- **左サイドバー（アウトライン）**: 見出しが追加されるとサイドバーのアウトラインに反映される。反映されない場合は適用が失敗している可能性がある
- **URL の `#heading=h.xxxxx`**: 見出し行をクリックすると URL にアンカーが付く。付かない場合はスタイル未適用

### スタイルの一括更新（太文字化など）

特定の見出しスタイルにカスタム書式（太文字など）を適用し、同スタイルの全箇所に一括反映する手順。

1. 対象の見出し行を `Home → Shift+End` で選択
2. 書式を適用（例: `Cmd+B` で太文字）
3. ツールバーのスタイルドロップダウンを開く
4. "Heading 1" にホバーしてサブメニューを表示
5. "Update 'Heading 1' to match" をクリック → ドキュメント内の全 H1 に反映

### よくあるミス

- **空行に適用してしまう**: ズームで確認 → Down キーで修正
- **前の行のスタイルが残っている**: `Cmd+Alt+0` で Normal text にリセットしてから再適用
- **サイドバーに表示されない**: Normal text のまま（スタイル未適用）なので、改めてクリックして確認・再適用する

## コンテンツの貼り付け

### クリップボードへのコピー

```bash
# ファイルからクリップボードにコピー
pbcopy < /path/to/file.txt

# コピー内容の確認
pbpaste | head -5
```

### 貼り付け手順

1. `pbcopy` でクリップボードにセット
2. Google Docs のタブをアクティブにしてクリック
3. `Cmd+A` で全選択（既存コンテンツを置き換える場合）
4. `Cmd+V` で貼り付け

### 注意事項

- クリップボードに別の URL や文字列が残っている場合があるため、貼り付け前に `pbpaste | head -5` で内容を確認する
- Markdown の記号（`#`, `**`, `-` 等）はそのままプレーンテキストとして貼り付けられる（Markdown レンダリングはされない）

## ドキュメント作成

### Google Drive への新規作成

1. 対象フォルダを Chrome MCP で開く
2. 「＋新規」→「Google ドキュメント」をクリック
3. タイトルをクリックして変更
