# Amazon (amazon.co.jp)

## 検索 URL

```
https://www.amazon.co.jp/s?k={検索キーワード}&rh=p_85%3A3169077051&sort=price-asc-rank
```

| パラメータ | 意味 |
|---|---|
| `rh=p_85%3A3169077051` | Prime 対象のみでフィルター |
| `sort=price-asc-rank` | 価格安い順でソート |
| `i=hpc` | ヘルス＆ビューティーカテゴリに絞る（必要に応じて追加） |

Prime フィルターで結果が少ない場合は、フィルターなしで検索してから配送日を手動確認する。

## 配送日の確認

- 検索結果の配送表示はあくまで目安。**必ず商品ページを開いて実際の配送日を確認すること**
- 人気商品（スコッティ等）は在庫薄・配送遅延が多いため特に注意
- 配送条件付き（「¥3,000以上で当日配送」等）の商品は注文金額に気をつける

## 価格表示

- 「¥X/メートル」などの Amazon の単価表示は概算。**正確な単価は自分で計算すること**
- 大量ケース購入ほど単価が安い傾向があるが、保管スペースと要相談
