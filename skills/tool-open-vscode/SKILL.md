# tool-open-vscode

現在の作業ディレクトリをVSCodeで開き、ユーザーが自分の目で変更内容(diff)を確認できるようにする。

## 実行手順

1. 現在の作業ディレクトリで以下を実行する。

```bash
code .
```

2. `code` コマンドが見つからない(`command not found`)場合は、以下にフォールバックする。

```bash
open -a "Visual Studio Code" .
```

3. VSCodeを開いたことをユーザーに一言伝える。

## 注意点

- ディレクトリを開くだけで十分。特定のファイルやビューを指定して開く必要がある場合(diffビューの直接指定など)は、このスキルの範囲外なので都度対応する。
- `code` コマンドの有無は環境によって異なるため、フォールバックを省略しない。
