# 実行が禁止されているコマンド

以下のコマンドはsettings.jsonのpermissions.denyで実行が禁止されている。

| コマンド | 理由 |
| --- | --- |
| `git reset` | ステージングやコミット履歴を巻き戻し、未コミットの変更やコミット自体を消失させうる |
| `git checkout` | ブランチ切り替えやファイル復元(`git checkout -- <file>`)でワークツリー上の変更を消失させうる |
| `git restore` | ワークツリーやステージングの変更を破棄する |
| `git clean` | 未追跡ファイル・ディレクトリを削除する |

## 実行が必要な場合の対応

上記コマンドはpermissions.denyの対象であり、"!"プレフィックスによるユーザ実行の提案もdenyルールを通過できずエラーになることが多い。
そのため実行が必要な場合は"!"での提案はせず、ユーザ自身のターミナルで実行してほしいコマンドの一覧をそのままユーザがコピーしやすいように明記して提示すること（改行したりせず、1行で提示すること）

## 注意

settings.jsonのpermissions.denyを変更した場合は、このファイルの内容も合わせて更新すること。
